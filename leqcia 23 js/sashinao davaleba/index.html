დავალება 4.1

    • let arr = [1, 2, 3, "ოთხი", "ხუთი"];
    alert(arr.length);

დავალება 4.2

  • მოცემულია მასივი arr. ინდექსის გამოყენებით მასივს დაამატეთ ახალი ელემენტი 'd', ხოლო რიგით მეორე ელემენტი ჩაანაცვლეთ 'x'-ით.

    • let arr = ['a', 'b', 'c'];
    arr[3] = 'd';
    arr[2] = 'x';
    alert(arr);

დავალება 4.3

  • მოცემულია ცარიელი მასივი arr. მეთოდი .push-ის გამოყენებით მასივს დაამატეთ ახალი ელემენტები 'a', 'b' და 'c'.

    • let arr = [];
    arr.push('a', 'b', 'c');
    alert("V1 arr = " + arr);

    arr.push(1,2);
    alert("V2 arr = " + arr);

    alert(typeof arr);

დავალება 4.4

  • მოცემულია მასივი arr და ცვლადები key1, key2. key1-ის ინდექსის მქონე მასივის ელემენტს მიუმატეთ key2 ინდექსის მასივის ელემნტი და შედეგი კონსოლზე გამოიტანეთ.

    • let arr = [1, 2, 3, 4, 5], key1 = 1, key2 = 2;
    alert("V1 = " + parseInt(arr[key1] + arr[key2]));

    arr.push(6,7,8);
    arr.unshift(-1,0);
    alert("V2 = " + arr);
